PathFS: File System for Edge Computing!

  • Powerful

    PathFS offers handy file system facilities! Easily create, move, and remove directories and files as if it is a central filesystem! Replication, consistency, and conflict resolution is handled by PathFS.

  • Streaming

    PathFS supports real time streaming! Live data from cameras, thermometers, and other sensors can quickly be streamed to the various receivers!

  • PathStore

    PathFS is built on top the PathStore, a general-purpose storage system for edge computing! PathFS has optimized PathStore and made it more suitable for low-latency streaming!

  • Hierarchical

    In edge environments, tasks are divided between cloud, network, and edge nodes. As we go down the hierarchy, nodes have much lower network latency, at the expense of lower computing capacities. PathFS' hierarchical design takes the most out of this situation as each node stores only a subset of its parent's data!
